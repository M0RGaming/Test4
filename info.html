Step 1:

Boot up the Board, (press the power button). If after 10 seconds, lights do not turn on to automaticaly warm up, then refer to the next 2 steps (Step 2 and Step 2.5). If they do turn on, wait for 5 minutes, WITHOUT TOUCHING THE BOARD. There will be a count down just above the command line.

Step 2:

Look at the Cues monitor, if it has 2 cues (0.1 and 0.2) that go under the name [Warm up] and [Blackout], then press the play button. It will automatically warm up the lights, and set it back to blackout after 5 minutes, so feel free to go and do something else (stay in the same room).

Step 2.5:

If there is not 2 cues (0.1 and 0.2) bring the grandmaster down fully, type [1] [Thru] [5] [0] [0] [Full] [Enter]. Then Bring up the grandmaster up to about 30%. Wait for about 5 minutes, then type [1] [Thru] [5] [0] [0] [Out]. Finally Bring up the grandmaster up to 100% again.

Faders/Channels:

To change the intensity of a singular channel, type the channel number followed by [At] then what you want the intensity to be (Followed by an enter). For example lets say you want channel 5 to have an intensity of 50%. You would type [5] [At] [5] [0] [Enter].

Faders/Channels Pt.2:

To change the intensity of multiple channels at once, type the lowest channel you want, followed by [Thru], then the highest channel. Then type [At] and the intensity you wish it to be. Finally, press the [Enter]. For example, if you want to set channels 3-7 to 84% intensity, you would type in: [3] [Thru] [7] [At] [8] [4] [Enter].

Faders/Channels Pt.3:

You are always able to use it the old (but simple) way, by using the faders. Turn the knob at the top to say [Channels 1-40] (or if you are using other channels, just set it to that range), then simply move up and down the sliders to change the intensity.

Memories:

Instead of memories, we will be using the Submasters which are basically memories. Submasters will store any input you give to them, and will allow for easy recall. In order to create a submaster, first set up the input that you want to save. Then press [Record] [Sub] then the number of the submaster you wish to save it to. If you wish to have easy access to it save it to one of the first 40. By saving it to one of the first 40 you are able to switch the fader mode to submaster and use the sliders to control the submasters. If you choose over 40, you will have to use the input command [Sub] followed by the submaster number, then the intensity of the submaster. For example lets say you want to pull up the submaster 61 at 75% intensity. You would input the commands [Sub] [6] [1] [At] [7] [5] [Enter].

Macros:

If you've played computer games before you probably already know what a macro is. If you don't, then a macro is a sequence of inputs that are stored. Unlike a memory/submaster, macros take raw inputs rather then intensity. To put it into perspective, a submaster takes the result and saves it, where as the macro takes the process. Using macros you are able to store a long command and condense it down into 3 button presses, [Macro] followed by the macro number, then [Enter]. For example, in order to activate the macro in slot 5, you would type [Macro] [5] [Enter].

Macros Pt.2:

To save a macro, you can type [Learn] [Macro] then the slot you would like the macro to take up, then [Enter]. If above the command line there is a blinking red Learning Macro, then its recording. Next type in whatever you wish it to do (don't forget to press enter) You can add more then 1 command if you so wish. When you are done typing all you want, then press learn, and the blinking red text should go away. For example, If you wish to save the command [1] [Thru] [50] [Full] [Enter], in macro slot 4, then you would type, [Learn] [Macro] [4] [Enter] [1] [Thru] [50] [Full] [Enter] [Learn].

Macros Pt.3:

It is possible to edit macros, simply head to the macro page, select the macro you would like to edit, then single click on the macro contents. Then use the arrow keys to move between terms. If you need help please contact someone who has experience working with macros (most expert lighting operators will).

Cues:

Remember the stack function on the old board (ETC SmartFade)? No, well thats ok, as it was REALLY REALLY obscure. Basically it allowed you to create a sequence of memories. Cues are effectively the same, except much easier to use. To save a cue, set up the input, as you would for a memory, then press [Record] [Cue] then the cue number you wish to save it as (don't forget an [Enter]). If you save more than one, by pressing the play button it will go from your current cue, to the next highest cue. Finally to go to a certain cue, you would type in [Go To Cue] then the cue number, then [Enter].

Advanced Features:

Memories, Macros, and Cues have many advanced features, that take a bit more work to pull off, such as auto turning off memories, and macros being capable of almost anything (as they are pretty much just creating and running a script (like programming)) (even running your show for you!), and linking cues together so that they autoplay. If you would like to learn about these advanced features then speak to one of us people who have experience messing around on the board, or read the manual, as it is too hard to simply type up (even though the manual is typed up, its somewhere around 300 pages).

Shows:

Typically when being in charge of lighting for a show, you would create a new show, which is basically a save file. You can create one by pressing the up arrow right of the command line, clicking on File, then double click New. Keep the patch 1 to 1 selected, and click OK. Before you do this though, remember to save the show that was previously open. To open a pre existing show, go to file, then click Open. Find the show file you wish to open and double click on it. To save a show, well go to File, and press save.
